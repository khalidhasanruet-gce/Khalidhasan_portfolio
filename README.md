# Md Khalid Hasan - Deployable Academic Portfolio

This is a static personal academic website prepared for direct deployment.

## Files
- index.html
- style.css
- script.js
- Hasan_CV.pdf

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Set source to the main branch root.
5. Save. Your site will be live within a few minutes.

## Deploy on Netlify
1. Go to Netlify.
2. Drag and drop this folder or ZIP file.
3. Netlify will publish the site automatically.

## Deploy on Vercel
1. Create a new project.
2. Upload this folder.
3. Since this is a static site, deployment is immediate.
